# Installation - SPEC-2

## Prerequisites

- Completion of SPEC-1, or use of `repo/spec-reference.md`.
- Copilot Chat with Plan Mode.
- Basic understanding of REST APIs and Flask.

## Setup

1. Open the `repo/` folder.
2. Read `repo/spec-reference.md`.
3. Create `repo/design.md`; this is where the design document will be saved.

## Verification

There is no command to run. The exercise is complete when `repo/design.md` contains the design, ADR and Mermaid flow.
